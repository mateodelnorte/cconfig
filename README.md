# cconfig

`cconfig` is a simple cascading config system

## Installation

This package is available on [npm][1] as: `cconfig`

    npm install cconfig

## Usage

    var config = require('cconfig')('./PATH/configuration.json');

`cconfig` will have current enviroment configuration

# Run the tests

    npm test

[1]: https://www.npmjs.org/package/cconfig
